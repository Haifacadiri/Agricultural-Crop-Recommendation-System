# 🌾 Crop Prediction & Price Estimation System

## 📌 Project Overview
This project is an intelligent agriculture-based system that helps farmers and stakeholders make better decisions using Machine Learning.

It predicts:
- ✅ The most suitable crop based on environmental conditions  
- 💰 Future price trends of crops for better planning  

The system aims to improve productivity, reduce risk, and increase profitability in agriculture.

---

## 🚀 Features

### 🌱 Crop Prediction
- Predicts the best crop to grow based on:
  - Temperature
  - Humidity
  - Rainfall
  - Soil nutrients (N, P, K)
- Uses Machine Learning model for accurate results

### 📊 Crop Price Estimation
- Predicts crop prices for upcoming months
- Helps in decision making for selling and storage
- Provides month-wise price insights

---

## 🧠 Technologies Used
- Python  
- Machine Learning (Scikit-learn / Pandas / NumPy)  
- HTML, CSS, JavaScript  
- API Integration (Weather data)

---

## 📥 Inputs

### For Crop Prediction:
- Location  
- Temperature (°C)  
- Humidity (%)  
- Rainfall (mm)  
- Nitrogen (N), Phosphorus (P), Potassium (K)

### For Price Prediction:
- Crop Name  
- Month & Year  

---

## 📤 Outputs

- 🌾 Predicted Suitable Crop  
- 📈 Monthly Crop Price Forecast (12 months)
